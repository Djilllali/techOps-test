Documentation backend

Ceci constitue la partie backend du test technique, conçu pour évaluer les compétences en développement d'interfaces React, en API Node.js, et la capacité à gérer efficacement un fichier CSV. J'ai implémenté cette solution en utilisant Express.js en conjonction avec des outils tels que Multer pour la gestion des fichiers, csv-parse pour l'analyse des fichiers CSV, et Archiver pour la compression.

Allons-y!

Module de service pour la manipulation de fichiers CSV.

csvService

parseFile

Cette fonction prend un fichier CSV, le divise en données maleData et femaleData, convertit ces données en chaînes CSV distinctes, crée un fichier ZIP et renvoie le chemin du fichier ZIP créé.

parseFile(file: Object): Promise
Parameters
file (Object) Objet représentant le fichier CSV.
Returns
Promise: Renvoie le chemin du fichier ZIP résultant.
Throws
  • Error: Lance une erreur si l'analyse du fichier échoue.

convertArrayToCsvString

Cette fonction prend un tableau de données et le convertit en un CSV, en utilisant la bibliothèque csv-writer.

convertArrayToCsvString(dataArray: Array): Promise
Parameters
dataArray (Array) Tableau de données à convertir.
Returns
Promise: Renvoie le CSV résultant.

zippingService

Module de service pour la compression de données.

zippingService

createZipFile

Cette fonction crée un fichier ZIP à partir des données CSV masculines et féminines, en utilisant la bibliothèque archiver, puis renvoie le chemin du fichier ZIP créé.

createZipFile(filename: string, maleData: string, femaleData: string): Promise
Parameters
filename (string) Nom du fichier d'origine.
maleData (string) Données masculines au format CSV.
femaleData (string) Données féminines au format CSV.
Returns
Promise: Renvoie le chemin du fichier ZIP créé.
Throws
  • Error: Lance une erreur si la création du fichier ZIP échoue.
